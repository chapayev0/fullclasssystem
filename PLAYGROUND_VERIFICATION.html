üîß Troubleshooting

Issue: Database tables not created

Solution:
1. Make sure you ran: php helpers/setup_playground_tables.php
2. Check database connection in db_connect.php
3. Verify MySQL user has CREATE TABLE privileges

Issue: Images not uploading

Solution:
1. Check that assest/images/playground/ directory exists
2. Set directory permissions to 755 or 777
3. Verify PHP upload_max_filesize setting

Issue: Games not appearing

Solution:
1. Verify game status is set to "active"
2. Check category is marked as "active"
3. Ensure game is assigned to at least one category
